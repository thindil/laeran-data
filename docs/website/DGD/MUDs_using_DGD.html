What MUDs Use DGD?

You can find most of these DGD MUDs the same way I did -- go to the MUDConnector and search the MUDs on the keyphrase "DGD". Here's what I found when I did that search.

Skotos Games

Skotos games give one free trial month and are flat-rate per month afterward. One account gives access to the major Skotos games including (non-DGD-based) The Eternal City and Grendel's Revenge, and a variety of web games whose implementations I know nothing about. My understanding is that standard MUD clients work for Castle Marrach, but that you're far better off using the Java or ActiveX clients. For details, you should have a look at Skotos' web page.

  • Castle Marrach is the first Skotos game and features large amounts of role-playing and little or no killin' stuff.
  • Welcome Room is an out-of-character chat theatre.

2.4.5- or custom-based MUDs

  • Albion MUD at mud.veda.is port 4000
  • Asgard's Honor at betterbox.net port 5454
  • IgorMUD at igormud.org port 1701
  • Xyllomer at xyllomer.mud.de port 3000
  • Geas MUD at geas.franken.de port 3333
  • Electric Sheep at spg.franklin.edu port 2000 is run by Jay Shaffstall, a DGD list regular. He even uses some Phantasmal-based components!
  • Esforonin MUD is in Sweden, and is working on its own custom MUDLib. The MUD's primary language is English, and it's located at esforonin.net, port 1409.

LPMOO-based MUDs

  • Rob Leslie's LPMOO is used by MirrorMOO at http://mirror.moo.mud.org:8080/.

Melville-based or Phantasmal-based MUDs

  • (None currently)

If your DGD MUD or MUD library has been unjustly excluded, please send the same information as above, including your URL if any and the host and port. Feel free to send additional information as well. Information should be directed to angelbob at users dot sourceforge dot net. Thanks!
