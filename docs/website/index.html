Setting Up Your MUD

  • How do I get Phantasmal installed?
  • To get started with Phantasmal, check out our tutorials! They cover basic administration and how to build rooms, items, mobiles and so on.
  • Please read "How to set up a new Phantasmal-based MUD". This will get you started adding your own new code and features to your MUD.
  • You may want the Phantasmal function API for building a Phantasmal-based game. It's not as up-to-date as most of the documentation, but it's still frequently better than just reading the source code.
